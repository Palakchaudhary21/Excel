# Smartphone Usage & Productivity Analysis

A data analysis project that explores the relationship between smartphone usage patterns and productivity using dataset insights and visualizations.

## Overview
The **Smartphone Usage & Productivity Analysis** project focuses on understanding how smartphone habits impact productivity levels. The analysis helps in identifying trends, usage behavior, and their effects on daily efficiency.

Key insights include:
- Screen time vs productivity
- App usage patterns
- Daily usage behavior
- Productivity trends

## Project Objectives
- To analyze smartphone usage data
- To identify patterns affecting productivity
- To visualize relationships between usage and efficiency
- To derive meaningful insights from the dataset

## Tools & Technologies
- Microsoft Excel
- Data Cleaning & Preprocessing
- Data Visualization Techniques

## Key Features
- Cleaned and structured dataset
- Analysis of screen time

- Productivity comparison
- Data-driven insights
